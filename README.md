# 网上图书商城

## 项目简介
网上图书商城是基于Java Web、SSM框架、JSP页面、Mysql数据库的在线图书销售系统。分为前台系统和后台管理系统，前台系统就是图书商城，用户可以登录，选购自己所需的图书，加入购物车并支付。管理员可以登录后台系统对图书进行、订单、用户等数据进行管理。角色上分为：普通用户和管理员。
